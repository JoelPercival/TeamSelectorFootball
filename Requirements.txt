Requirements for Team Picker
- must be able to pick even teams 
- must be able to store the player ranking
- must be able to edit the player rankings
- should be able to save the final team selection
	-  could export it in a format which can be posted to the chat


Manual process currently used:
- Add active players to the spreadsheet
- blackout all players who arent playing
- create number of teams required
- add players in to each side to create equal valued teams
- ensure not equal spread of different style players
